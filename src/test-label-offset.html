🌍 城市标签偏移功能测试

✅ 功能实现完成

现在 labelOffset 已经直接集成到 CityData 接口中！

📝 新的使用方式

// 在 CityData 中直接定义 labelOffset const cityData: CityData[] = [ { name: '深圳', lng: 114.0579, lat: 22.5431, color: 0xffffff, labelOffset: { x: 0, y: 0, z: 0 } // 无偏移 }, { name: '北京', lng: 116.4, lat: 39.9, color: 0xffffff, labelOffset: { x: 5, y: 0, z: 0 } // 向右偏移 } ] // 使用新方式创建城市 const city = useCity(cityData[0], { onHover: (city) => console.log('悬停城市:', city.name) })

🔄 向后兼容

传统调用方式仍然支持：

// 传统方式仍然可用 const city = useCity([lng, lat], '城市名', { color: 0xffffff, labelOffset: { x: 5, y: 0, z: 0 } })

🎯 当前效果

在 http://localhost:3001/ 中查看效果：

  • 深圳：无偏移
  • 北京：向右偏移
  • 上海：向左偏移
  • 西安：向上偏移
  • 南京：向下偏移
  • 杭州：右上偏移
  • 东莞：左上偏移
  • 成都：更向上偏移
